Every "type of data that can be sent to this server"

Give Data:

TEMP        -   Temperature                 -   Float
HUMID       -   Humidity                    -   Float
ALARM       -   Alarm gone off              -   Bool
ARMED       -   Alarm is armed              -   Bool
LIGHTSTATUS -   Light is on or off          -   Bool
WINDOW      -   Window is opened or closed  -   Bool
SOUND       -   Sends the highest sound     -   Int


Recieve Data:

GETSOUND    -   Get the sound stored on the server  -   Int
